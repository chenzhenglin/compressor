# compassor

> 基于electron-vue的前端文件打包压缩的桌面应用软件
> v1.0.0实现了单个js的打包跟还原打包前源文件的功能

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

# run unit & end-to-end tests
npm test


```
